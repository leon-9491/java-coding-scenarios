Input Float Value
=================

Problem Description:
Write a program that reads a floating-point value from the user and prints it to the console. The program should use the Scanner class to capture the float input and then display the value exactly as entered. Ensure that your program correctly handles the input and output as specified.

Input Format:
A single floating-point value input by the user.

Output Format:
The floating-point value entered by the user, printed to the console.
